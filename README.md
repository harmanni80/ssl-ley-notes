# ssl-ley-notes

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面以静态 HTML 文件的形式存放，不依赖特定域名或后端服务，可直接通过浏览器打开查看。

## 项目简介

`ssl-ley-notes` 是一个轻量级的页面归档仓库，主要用途包括：

- 存放与 SSL/TLS 相关或无关的独立 HTML 文档
- 记录个人学习过程中的笔记、示例或工具页面
- 提供一个简洁的、可直接访问的静态页面集合

所有页面均为自包含的 HTML 文件，无需额外配置即可运行。

## 目录说明

```
ssl-ley-notes/
├── index.html          # 仓库首页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   └── example2.html
└── assets/             # 静态资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：所有独立 HTML 页面均放置于此目录下。
- `assets/`：存放页面所需的公共样式、脚本或媒体文件，方便统一管理。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可单独访问。
- 页面之间无强依赖关系，可根据需要自由增减。
- 页面内容可能涉及技术笔记、工具演示、学习记录等，不保证时效性或完整性。
- 归档内容仅供参考，不构成任何形式的建议或承诺。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/ssl-ley-notes.git
   ```
2. 直接打开 `pages/` 目录下的 HTML 文件，或在浏览器中通过文件路径访问。
3. 也可将仓库部署到任何静态托管服务（如 GitHub Pages）进行在线访问。
